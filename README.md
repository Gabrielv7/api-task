
# API REST de tarefas

Essa API foi desenvolvida para prática de conhecimentos, é uma API simples de usar, basta cadastrar uma categoria e uma tarefa condizente com a categoria cadastrada.

## Tecnologias

- Java 17
- REST
- Spring Framework 3
- Maven
- Banco de dados em memória H2
- Junit
- Mockito
- Jacoco
- Swagger
